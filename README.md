Este repositório documenta o processo de **Hardening** (recrudescimento de segurança) aplicado aos servidores da infraestrutura da **SkyBoom**. O projeto foca no gerenciamento seguro de identidades através do terminal, aplicação de algoritmos de hashing para credenciais administrativas e análise de superfícies de ataque contra vazamentos de arquivos de configuração.
